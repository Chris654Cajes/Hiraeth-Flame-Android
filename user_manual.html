Overview

Hiraeth Flame is a comprehensive media management application designed with a beautiful dark theme and purple accents. The app provides a seamless experience for managing photos, videos, and creating content.

Media Library

Browse, search, and filter your photos and videos with ease. Sort by date, name, or type.

Camera

Capture photos and record videos directly within the app with full camera controls.

Albums

Organize your media into custom albums for better organization and quick access.

Image Editor

Adjust brightness, contrast, saturation, rotate, and draw on your images.

Video Tools

Trim videos by setting start and end points, then export as new files.

Reel Studio

Combine multiple clips into a single reel video with ordered sequencing.

Getting Started

  1. Install & Launch

    Download and install Hiraeth Flame from the app store. Open the app to see the main interface with the bottom navigation bar.

  2. Grant Permissions

    On first launch, you'll be prompted to grant media and camera permissions. These are required for the app to access your photos, videos, and camera functionality.

  3. Navigate the App

    Use the bottom navigation bar to switch between three main sections: Library, Camera, and Albums.

💡 Tip

The app uses a dark theme with purple accents (#9900FF) to reduce eye strain and provide a modern, elegant experience.

Using the Camera

  1. Navigate to Camera

    Tap the Camera icon in the bottom navigation bar.

  2. Take a Photo

    Tap the Photo button to capture an image. The photo will be saved to your library.

  3. Record a Video

    Tap the Record button to start recording. Tap again to stop. Videos are automatically saved.

⚠️ Permission Required

Camera functionality requires camera permission. If you haven't granted permission, you'll see a prompt to do so.

Editing Tools

  • Brightness — Adjust the overall lightness or darkness of the image (0-120)
  • Contrast — Control the difference between light and dark areas (0-130)
  • Saturation — Modify the intensity of colors (0-200)
  • Rotate 90° — Rotate the image clockwise in 90-degree increments
  • Draw — Toggle drawing mode to sketch or annotate on your image
💡 Tip

All adjustments are applied when you tap "Apply & save". You can preview changes in real-time before saving.

How to Edit an Image

  1. Open Media Detail

    Tap on a photo in your Library to open its detail view.

  2. Launch Editor

    Tap the Open image editor button.

  3. Adjust Settings

    Use the sliders to adjust brightness, contrast, and saturation to your preference.

  4. Apply & Save

    Tap Apply & save to process and save the edited image.

Trimming a Video

  1. Select a Video

    Open a video from your Library to access its detail view.

  2. Open Video Tools

    Tap the Trim / video tools button.

  3. Set Start Point

    Use the Start % slider to set where the trimmed video begins (0-100%).

  4. Set End Point

    Use the End % slider to set where the trimmed video ends.

  5. Export

    Tap Export trim as new file to create a new video file with your selected segment.

💡 Tip

The trim operation only processes the video track. The original video remains unchanged — a new file is created.

Creating a Reel

  1. Access Reel Studio

    Navigate to Reel Studio from the app's main navigation or media actions.

  2. Select Clips

    Tap on clips in your library to select them. The order in which you tap determines the sequence.

  3. Stage the Reel

    Tap Stage reel (ordered copies) to prepare the clips for merging.

  4. Merge & Export

    The app will merge the staged clips into a single MP4 file using Media3 Transformer.

💡 Tip

Use Clear selection to reset your clip order and start over if needed.

Importing Media

  1. Tap Import Button

    From the Library screen, tap the floating action button (+) and select Import.

  2. Choose File

    Tap Choose image or video to open the file picker and select your media.

  3. Preview & Rename

    Preview the selected media and optionally give it a custom title/filename.

  4. Save to Library

    Tap Save to library to import the media into your collection.
